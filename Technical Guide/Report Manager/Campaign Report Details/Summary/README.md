**The summary** tab provides a brief summary of the campaign like how many user get the campaign email or how many of them clicked on the link in the email or how many people gave no response to the email. 

Here, in Summary section, you can also transfer all private and general details of the campaign into an excel file via **“Download Full Report”** option, or you can resend campaign by filtering it with the various options like sending the campaign to the users who gave no response etc.   See the following screenshot.
