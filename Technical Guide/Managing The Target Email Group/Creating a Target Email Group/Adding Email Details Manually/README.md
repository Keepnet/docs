# Adding email details manually

You can also add email details to your target groups manually. See the screenshot below.

![Adding the user manually](https://www.keepnetlabs.com/wp-content/uploads/Ekran-G%C3%B6r%C3%BCnt%C3%BCs%C3%BC-2018-07-23-18-15-41.png)

When you click on **Add User Manually**, now you will open a new window, **Email User Edit** which has to be filled in.

![Email user edit](https://www.keepnetlabs.com/wp-content/uploads/Email-user-edit-1024x521.png)

After filling in this section, and by clicking the Save button, you complete manual email adding process.
